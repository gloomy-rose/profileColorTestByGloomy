# 🎨 Profile Color Test

Ferramenta visual para testar combinações de cores em tempo real em um layout de perfil. 
A ideia é simples: facilitar a experimentação de cores **em contexto real**, permitindo visualizar rapidamente como fundo, textos e componentes se comportam juntos — sem precisar editar CSS manualmente o tempo todo.

---

## ✨ Funcionalidades

- 🎨 Alteração de cores em tempo real:
  - Background
  - Texto
  - Boxes (todos unificados)

- 🎯 Input manual em **Hexadecimal**
- 🧪 Color picker integrado

- 🖼️ Preview de imagem por URL
  - Imagem principal
  - Miniaturas

- 👤 Nome do personagem editável

- 📦 Presets rápidos de cores

- 📋 Extração de paleta atual:
  - Exibe cores organizadas
  - Botão para copiar facilmente

- 🧼 Toggle para esconder/mostrar painel de controles

---

## 💡 Objetivo

Esse projeto foi criado como uma ferramenta prática para quem:
- testa UI/UX
- monta layouts personalizados
- ou simplesmente precisa “sentir” se uma combinação de cores funciona

Porque às vezes… só vendo pra saber.

---

## 🚀 Como usar

1. Abra o arquivo `index.html` no navegador  
2. Ajuste as cores pelo painel  
3. Teste imagens e conteúdo  
4. Copie a paleta final
